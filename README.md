# Miguel Gallego — Portfolio

Personal portfolio built with [Astro](https://astro.build) and [Tailwind CSS](https://tailwindcss.com).

## Tech Stack

- Astro 7
- Tailwind CSS 4
- TypeScript

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:4321](http://localhost:4321).

## Build

```bash
npm run build
npm run preview
```

## Deploy

Ready for [Vercel](https://vercel.com). Connect the repository and deploy — no extra configuration needed.
